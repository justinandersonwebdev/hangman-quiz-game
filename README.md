# Classic Hangman Quiz Game!

Guess the secret word by selecting the remaining letters! Too many wrong guesses and you lose! (and so does the stick-man...)

## Getting Started

- Clone this repo and run the following terminal commands in the project root:

```sh
# install dependencies
yarn
```

```sh
# start your app in development mode and rebuild assets on file changes
yarn dev
```

- in your browser, navigate to the localhost port provided

```sh
# to build the app for production:
yarn build
```

```sh
# run the app in a preview of production mode:
yarn preview
```
